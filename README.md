# ModelCompass

ModelCompass is a production-ready financial governance pipeline built on the Databricks Lakehouse. It addresses the "Dual-Risk" of modern banking: Model Decay due to economic drift and AI Hallucination in customer communications.

Unlike static scoring models, ModelCompass is a living system that monitors its own accuracy, triggers automated retraining when the economy shifts, and audits LLM-generated loan explanations for regulatory compliance.
